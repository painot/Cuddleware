### Setup

```py
import requests
import types

url = "https://raw.githubusercontent.com/painot/Cuddleware/main/main.py"
code = requests.get(url).text

Cuddles = types.ModuleType("Cuddleware")
exec(code, Cuddles.__dict__)
```
